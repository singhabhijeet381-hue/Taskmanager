📝 Task Manager Web App

A **Flask-based Task Manager application** that helps users create, manage, and track their daily tasks efficiently. This project demonstrates full-stack web development using Python, Flask, and database integration.

---

## 🚀 Features

- ✅ Create, update, and delete tasks  
- 📋 View all tasks in a structured format  
- 🗂️ Organized backend using Flask Blueprints  
- 🧠 Form handling with Flask-WTF  
- 💾 Database integration (SQLite)  
- 🎨 Clean UI with Jinja templates

- 🛠️ Tech Stack

- **Backend:** Python, Flask  
- **Database:** SQLite  
- **Forms:** Flask-WTF  
- **Frontend:** HTML, CSS, Jinja2  
- **ORM:** SQLAlchemy
